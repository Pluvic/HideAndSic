# HideAndSic

## Description
HideAndSic is a web application designed to analyze suspicious files for hidden content, malware, or other security threats. It provides users with tools to upload files and receive detailed reports on their analysis.

## Features
- File upload functionality
- Comprehensive analysis of file contents
- Detection of hidden data and malware
- User-friendly interface for viewing analysis reports
- Support for multiple file types

## Installation
1. Clone the repository:
    ```bash
    git clone "https://github.com/Pluvic/HideAndSic"
    ```
2. Navigate to the project directory:
    ```bash
    cd HideAndSic
    ```
3. Load the docker images:
    ```bash
    docker-compose up -d --build
    ```
4. Access the web application at `http://localhost:3000`
5. To stop the application, run:
    ```bash
    docker-compose down
    ```
## Usage
1. Open your web browser and go to `http://localhost:3000`.
2. Use the file upload feature to select a suspicious file from your computer.
3. Click the "Analyze" button to start the analysis process.
4. Wait for the analysis to complete and view the detailed report generated by the application.
5. Review the findings and take appropriate action based on the report.
